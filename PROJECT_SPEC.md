# FLIT 프로젝트 기획서

> 플리마켓·푸드트럭 셀러를 위한 행사 정보 및 커뮤니티 플랫폼

---

## 1. 서비스 개요

**FLIT**은 플리마켓·푸드트럭 셀러가 행사 정보를 쉽게 찾고, 주최사의 평판을 검증하며, 수익성 높은 행사를 선택할 수 있도록 돕는 구독형 플랫폼이다.

- **핵심 가치**: 셀러 중심의 투명한 행사 정보 제공
- **타깃 유저**: 플리마켓 셀러, 푸드트럭 운영자, 행사 주최사
- **기술 스택**: Next.js (App Router), Supabase (PostgreSQL + Auth)

---

## 2. 사용자 유형 및 멤버십 플랜

| 플랜 | 대상 | 월 요금 | 주요 기능 |
|------|------|---------|----------|
| `free` 일반회원 | 모든 가입자 | 무료 | 커뮤니티 열람, 모집공고 열람, 리뷰 작성 무제한 |
| `flea_market` 플리마켓 | 플리마켓 셀러 | 9,900원 | 모집공고 무제한 지원, 리뷰 무제한, 플리 전용 정보 |
| `foodtruck` 푸드트럭 | 푸드트럭 운영자 | 19,900원 | 플리 모든 혜택 + 푸드트럭 전용 공고, 매출·집객 분석 |
| `organizer` 주최사 | 행사 주최 단체 | 99,000원 (별도문의) | 모집공고 등록, 지원자 관리, 행사 홍보 페이지 |

**리뷰 열람 권한**
- `free`: 리뷰 상세 열람 불가
- `flea_market`: 일반 셀러(seller) 리뷰만 열람 가능
- `foodtruck` / `organizer`: 모든 리뷰 열람 가능

## 3. 핵심 기능

### 3-1. 홈 (`/`)
- 히어로 배너 (캐러셀, 3개 슬라이드)
  - 리뷰 작성 유도 (첫 달 무료 프로모션)
  - 수익 데이터 행사 검색 유도
  - 행사 개최 문의 유도
- 추천 행사 목록
- 최근 리뷰 목록
- 모집 중인 공고

### 3-2. 행사 검색 (`/search`)
- 탭: 행사 / 리뷰 / 주최사
- 필터: 지역, 행사 유형, 참가비, 날짜
- 정렬: 최신순 / 참가비 낮은순·높은순 / 평점 높은순 / 리뷰 많은순 등
- 무한 스크롤 (10개씩)
- 구독 잠금(Lock) 처리

### 3-3. 행사 상세 (`/events/[id]`)
- 행사 기본 정보, 주최사 정보
- 리뷰 목록 (구독 플랜별 열람 권한)
- 스크랩(북마크) 기능

### 3-4. 주최사 (`/organizers/[id]`)
- 주최사 프로필, 평점, 통계
- 주최한 행사 목록
- 셀러 리뷰 목록

### 3-5. 모집공고 (`/recruitments/[id]`)
- 공고 상세 (날짜, 장소, 참가비, D-Day)
- 평점 항목별 분석 (수익성, 집객, 운영)
- 리뷰 목록
- 지원하기 버튼 (구독자 전용)

### 3-6. 리뷰 작성 (`/reviews/write`)
- 행사 검색 후 선택
- 셀러 유형 선택 (플리마켓 / 푸드트럭)
- 항목별 별점: 수익성, 집객도, 운영/소통, 시설/환경, 재참가 의향
- 텍스트 후기 작성
- 공개 범위 설정 (모두공개 / 구독자공개)
- 모든 셀러 무제한 작성 가능

### 3-7. 커뮤니티 (`/community`)
- 게시글 목록 (카테고리 필터)
- 게시글 상세 (`/community/[id]`) - 댓글 포함
- 게시글 작성 (`/community/write`)

### 3-8. 마이페이지 (`/mypage`)
- 내 정보 / 구독 플랜 확인
- 내 리뷰 목록
- 관심 행사 목록 (스크랩)
- 내 게시글 목록
- 프로필 편집
- 설정

### 3-9. 구독 (`/subscribe`)
- 플랜별 비교 및 결제

### 3-10. 문의 (`/contact`)
- 행사 개최 의뢰 문의 폼

## 4. 데이터베이스 구조 (V2)

```
profiles          -- 사용자 (plan, seller_type 포함)
organizers        -- 주최사
base_events       -- 행사 기본 정보 (반복 행사의 원본)
event_instances   -- 행사 회차 (날짜·장소별 인스턴스)
recruitments      -- 모집공고 (event_instance에 연결)
reviews           -- 셀러 리뷰 (event_instance 기준)
scraps            -- 스크랩(북마크)
posts             -- 커뮤니티 게시글
post_comments     -- 커뮤니티 댓글
```

- `base_events + event_instances` 분리 구조: 같은 행사의 여러 회차를 별도 관리
- 평점 항목: `profit_rating`, `visitors_rating`, `operation_rating`, `facilities_rating`, `revisit_rating`
